# Script de Data Masking - Ambiente de Teste

Exemplo conceitual de migração segura com mascaramento de PII para Firestore.

## Como rodar localmente

**Pré-requisitos:** Docker e Node.js 18+

1. **Subir emulador do Firebase:**
```bash
docker-compose up -d
```
2. **Instalar dependências:**
```bash
npm install
```
3. **Executar scripsts:**
```bash
npm start
```
